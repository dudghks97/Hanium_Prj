# 개발 우선순위(초안)
0. 사진/동영상 데이터 ---> 앱
   - 갤러리 접근
   - 카메라 접근

1. 마스크 추천 ---> 구매
   - 얼굴 인식 => 얼굴형, 퍼스널 컬러 판별
   - 추천 : 판별 결과를 통한 추천
   - 웹크롤링 : 마스크 정보

2. 마스크 가상 피팅
   - 가상 피팅 : 앱 시작 시(최초 실행 시)
   - 얼굴 인식 : 얼굴 특징 값 가져오고
   - 피팅할 마스크 : 크롤링 후 3D 렌더링 하는 방식 예상
   - 가상 피팅 : 특징 값에 맞게 마스크를 씌워주기만

# 차별화 관련 아이디어
1. 커뮤니티
   - 사용자간 실시간 채팅
   - 실시간 코로나, 황사, 미세먼지 정보

2. 마스크 악세사리
   - 마스크 줄, 마스크 케이스 등 다양한 마스크 악세사리 추천

3. 사용자 입력을 통한 구매일 알림
   - 구매일, 구매 예정일, 수량 입력
   - 구매 예정일이 가까워지거나, 당일이 되면 알림

4. 퍼스널 컬러
   - AI 추천
   - 선택지를 통한 자가진단
